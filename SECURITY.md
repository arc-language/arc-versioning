# Security Policy

## Supported versions

| Version | Supported |
|---|---|
| 0.1.x | ✅ Yes |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email **kobecuppens@gmail.com** with:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You'll receive a response within 48 hours. We'll coordinate a fix and disclosure timeline together.
